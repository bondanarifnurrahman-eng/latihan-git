Ada dua cara 2 untuk menghubungkan Git lokal dengan GitHub, yaitu dengan HTTPS dan SSH.
1. **Menggunakan HTTPS:**

   - Pertama, pastikan Git sudah terinstal di komputer Anda.
   - Buka terminal atau command prompt.
   - Konfigurasikan username dan email Git Anda (jika belum dilakukan):
     ```bash
     git config --global user.name "YourUsername"
     git config --global user.email "<EMAIL>"
     ```
    - Inisialisasi repository Git di folder proyek Anda (jika belum dilakukan):
      ```bash
      git init
      ```   
    - Tambahkan remote repository GitHub:
      ```bash
      git remote add origin https://github.com/YourUsername/YourRepository.git (link repo github kamu)
        ```
    maka kamu sudah menghubungkan git lokal dengan github menggunakan https

Cara melihat remote repository yang sudah ditambahkan:
```bash
git remote -v
```
cara keluar dari file cd .. untuk keluar dari file di terminal
cara menghapus remote repository yang sudah ditambahkan:
```bash 
git remote remove origin
```
Cara mengganti URL remote repository:
```bash
git remote set-url origin https://github.com/YourUsername/YourRepository.git (link repo github kamu)
``` 
mkdir nama_folder_baru untuk membuat folder baru di terminal
